\section{Part A}\label{A}
\subsection{Designing the Motion Model}\label{motion}
The subject of the particle filter is a mobile robot moving on a 2D plane. Access is not given
to this robot's low-level controls (i.e differential drive actuation), so it is reasonable to
assume a unicycle model whereby motion can be achieved by a combination of linear and angular
velocity, thus mapping the robot as a 3 degree of freedom system represented by coordinates
$x$, $y$, and $\theta$. Hence, the motion model used for this filter is a nonlinear model
mapping $x_t$, $y_t$, and $\theta_t$ as follows for each next iteration of time elapsed $dt$:
\begin{enumerate}
\item If $w$ is zero:
\begin{equation}
x_t = x_{t-1} + v *cos(\theta_{t-1}) * dt + \epsilon_x
\end{equation}
\begin{equation}
y_t += y_{t-1} + v *sin(\theta_{t-1}) * dt + \epsilon_y
\end{equation}
\item If $w$ is nonzero, we represent motion as a circular trajectory:
\begin{equation}
x_t += x_{t-1} + \frac{v}{w} * (-sin(\theta_{t-1}) + sin(\theta_{t-1} + w*dt)) +
        \epsilon_x
\end{equation}
\begin{equation}
y_t += y_{t-1} + \frac{v}{w} * (cos(\theta_{t-1}) - cos(\theta_{t-1} + w*dt)) +
        \epsilon_y
\end{equation}
\begin{equation}
\theta_t = \theta_{t-1} + w*dt + \epsilon_{\theta}
\end{equation}
\end{enumerate}
\begin{equation}
\label{x_update}
x_t=x_{t-1}+v(cos(x))*dt
\end{equation}
\begin{equation}
\label{y_update}
y_t=y_{t-1}+v(sin(x))*dt
\end{equation}
